# Hack The Box

Writeups for Hack The Box machines are available here, organized into [<mark style="color:blue;">**Windows**</mark>](windows/)**,** [<mark style="color:blue;">**Linux**</mark> ](linux/)and[ <mark style="color:blue;">**Active Directory**</mark>](active-directory/) categories.
